This is a tiny purescript package to get you the Global JavaScript Context.
It exposes one function, which is `getContext`.

```purescript
getContext >>= \context ->
  -- In a browser
  context == window
  -- In Node.js
  context == global
  -- In a WebWorker
  context == self
```
